План курса

В курс входит изучение основ HTML/CSS и веб-верстки, работа с макетом, немного JavaScript, использование Github Pages в качестве хостинга

Программа постоянно дорабатывается и может отличаться от приведенной ниже.

Программа курса

    1. Проблемы образования. Как эффективнее учиться
    2. Как устроен Интернет
    3. Начало HTML (язык, синтаксис)
    4. Редакторы кода
    5. Первые элементы (p, h1-h6, a, img)
    1. Вложенность
    2. Форматирование текста (em, strong, q, blockquote)
    3. Форматирование данных (списки, таблицы)
    4. Начало CSS (первые свойства, стили браузеров)
    1. Ввод и отправка данных: формы и кнопки
    2. Инструменты разработчика, DOM
    3. Каскадность, специфичность, наследование
    4. Селекторы
    5. Стили по умолчанию
    6. Цвет
    1. Основные раскладки, display
    2. Позиционирование
    3. Box Model, отступы
    1. Планирование, работа с макетом в Фигме
    2. Семантические элементы (header, main и footer)
    3. Навигация (nav)
    1. Flexbox
    2. Создание нескольких страниц
    3. Ссылки между страницами
    1. Адаптивный дизайн
    2. Медиа-выражения
    3. Векторные изображения
    4. Подключение и редактирование SVG
    1. Из чего состоят сайты (critical rendering path)
    2. Начало JavaScript
    3. Написание JS скрипта для кнопки меню
    1. Типы сайтов по методу сборки. Серверы
    2. Использование Github Pages в качестве сервера
    3. Онлайн сервисы для сайта (проверка ошибок, тест скорости, подключение анализа)
    1. Вставка видео и интерактивных карт
    2. Что нужно знать про SEO
    3. Фавиконки, генераторы, форматы
    4. Ответы на вопросы
    5. Продолжение обучения
